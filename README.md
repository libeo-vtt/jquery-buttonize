# VTT jQuery Buttonize Plugin
Transform any HTML element into a button.

## Usage

1. Include jQuery:

	```html
	<script src="path/to/jquery.min.js"></script>
	```

2. Include plugin's code:

	```html
	<script src="path/to/jquery.buttonize.js"></script>
	```

3. Call the plugin:

	```javascript
	$("#element").buttonize({
		// config
	});
	```
